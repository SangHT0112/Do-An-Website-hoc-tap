Giới thiệu ngôn ngữ C++

C++ là một ngôn ngữ lập trình mạnh mẽ và phổ biến. Nó đặc biệt thích hợp cho việc phát triển phần mềm hệ thống, trình biên dịch, và các ứng dụng hiệu suất cao.

C++ được phát triển vào đầu những năm 1980 bởi Bjarne Stroustrup tại Bell Labs.

Tại sao nên học C++?

  • C++ là nền tảng của nhiều ngôn ngữ lập trình hiện đại như Java, C#, và có ảnh hưởng lớn đến nhiều ngôn ngữ khác.
  • Cung cấp khả năng kiểm soát cao cấp đối với phần cứng và bộ nhớ cùng với các tính năng lập trình hướng đối tượng.
  • Chạy trên nhiều nền tảng khác nhau (Windows, Linux, Unix, Mac OS X, v.v.).
  • Hiệu suất cao và thường được sử dụng cho các ứng dụng yêu cầu tài nguyên hạn chế và hiệu suất cao.
  • Được hỗ trợ rộng rãi và có nhiều tài liệu học tập từ cộng đồng.
  • Rất phù hợp để học các khái niệm lập trình cơ bản và nâng cao, bao gồm cả lập trình hướng đối tượng và lập trình tổng quát.

Biến trong ngôn ngữ lập trình C++

Các biến trong ngôn ngữ lập trình C++ là "thùng chứa" để lưu trữ thông tin.

Tạo (Khai báo) Biến trong C++

Trong ngôn ngữ lập trình C++, bạn cần khai báo biến trước khi sử dụng:

Ví dụ

int x = 5;

char y = 'A';


Trong ví dụ trên, biến x sẽ lưu trữ giá trị 5, và biến y sẽ lưu trữ giá trị 'A'.

Lưu ý: Khi bạn gán giá trị văn bản cho một biến trong ngôn ngữ lập trình C++, bạn cần sử dụng dấu nháy đơn ('') thay vì dấu nháy kép ("").

Lưu ý: Bạn cần khai báo biến trước khi sử dụng trong ngôn ngữ lập trình C++.

Hãy coi các biến trong ngôn ngữ lập trình C++ như các thùng chứa để lưu trữ dữ liệu.



Tìm hiểu chuyên sâu về biến trong C++

Một biến trong ngôn ngữ lập trình C++ có thể có tên ngắn hoặc tên mang tính mô tả hơn.

Quy Tắc Biến Trong ngôn ngữ lập trình C++

  • Biến trong C++ bắt đầu bằng ký tự chữ cái hoặc dấu gạch dưới.
  • Tên biến trong C++ không được bắt đầu bằng một số.
  • Tên biến trong C++ chỉ chứa các ký tự chữ cái, số và dấu gạch dưới (A-z, 0-9, và _).
  • Tên biến trong C++ không phân biệt chữ hoa và chữ thường.

Hãy nhớ rằng tên biến trong ngôn ngữ lập trình C++ không phân biệt chữ hoa chữ thường.

Biến đầu ra trong C++

Câu lệnh cout trong ngôn ngữ lập trình C++ thường được sử dụng để xuất dữ liệu ra màn hình.

Ví dụ sau đây sẽ chỉ ra cách xuất văn bản và giá trị của một biến trong ngôn ngữ lập trình C++:

Ví dụ

std::string txt = "W3Schools.com";

std::cout << "I love " << txt << "!";

Chuỗi trong ngôn ngữ lập trình C++


Chuỗi trong ngôn ngữ lập trình C++ là một chuỗi các ký tự, chẳng hạn như "Hello World!"

String

Các chuỗi trong ngôn ngữ lập trình C++ được bao quanh bởi dấu nháy kép.

Ví dụ

Tạo một biến kiểu string và gán cho nó một giá trị:

std::string greeting = "Hello";

Lưu ý: Trong ngôn ngữ lập trình C++, dấu nháy kép được sử dụng để xác định chuỗi ký tự.


Để sử dụng chuỗi, bạn phải thêm một tệp tiêu đề bổ sung vào mã nguồn, thư viện <string> :

Ví dụ

Đoạn mã dưới đây bao gồm thư viện string và tạo một biến kiểu string:

// Include the string library
#include <string>

// Create a string variable
std::string greeting = "Hello";

Nối string C++

Toán tử + có thể được sử dụng giữa các chuỗi để cộng chúng lại với nhau nhằm tạo thành một chuỗi mới.

Ví dụ

Đoạn mã dưới đây tạo các biến string và nối chúng lại với nhau:

std::string firstName = "John ";

std::string lastName = "Doe";

std::string fullName = firstName + lastName;

std::cout << fullName;

String Length

Để lấy độ dài của chuỗi, hãy sử dụng hàm length():

Ví dụ

Đoạn mã dưới đây tạo một biến string và xuất độ dài của nó:

std::string txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";

std::cout << "The length of the txt string is: " << txt.length();

Mẹo: Bạn có thể thấy một số chương trình C++ sử dụng hàm size() để lấy độ dài của chuỗi. Đây chỉ là bí danh của length(). Hoàn toàn tùy thuộc vào bạn nếu bạn muốn sử dụng length() hoặc size():

Ví dụ

Đoạn mã dưới đây tạo một biến string và xuất độ dài của nó:

std::string txt = "ABCDEFGHIJKLMNOPQRSTUVWXYZ";

std::cout << "The length of the txt string is: " << txt.size();

Toán tử C++

Toán tử được sử dụng để thực hiện các thao tác trên các biến và giá trị.

Trong ví dụ bên dưới, chúng tôi sử dụng toán tử + để cộng hai giá trị lại với nhau:

Ví dụ

int = 100 + 30 ;

Mặc dù toán tử + thường được sử dụng để cộng hai giá trị với nhau, như trong ví dụ trên, nhưng nó cũng có thể được sử dụng để cộng một biến và một giá trị hoặc một biến với một biến khác:

Ví dụ

Sử dụng các biến để tính tổng:

int sum1 = 100 + 50; // 150 (100 + 50)

int sum2 = sum1 + 250; // 400 (150 + 250)

int sum3 = sum2 + sum2; // 800 (400 + 400)

Hàm trong C++

Điểm mạnh thực sự của C++ đến từ khả năng sử dụng hàm.

C++ có một số hàm tiêu chuẩn được tích hợp sẵn và bạn cũng có thể tự định nghĩa các hàm tùy chỉnh của riêng mình.

Các hàm tiêu chuẩn trong C++

C++ cung cấp nhiều hàm tiêu chuẩn có thể gọi trực tiếp từ bên trong chương trình để thực hiện các tác vụ cụ thể.

Vui lòng tham khảo tài liệu C++ của chúng tôi để có cái nhìn tổng quan đầy đủ về các hàm tiêu chuẩn của C++.

Các hàm do người dùng xác định trong C++

Ngoài các hàm tiêu chuẩn, bạn cũng có thể tự định nghĩa các hàm của riêng mình trong C++.

  • Hàm là một khối lệnh có thể tái sử dụng nhiều lần trong chương trình.
  • Một hàm sẽ không tự động thực thi khi chương trình được tải.
  • Một hàm sẽ được thực thi thông qua lệnh gọi hàm.

Tạo một hàm

Định nghĩa hàm do người dùng bắt đầu bằng từ khóa void hoặc kiểu dữ liệu của giá trị trả về (nếu có), tiếp theo là tên của hàm:

Ví dụ

Hàm C++ myMessage() in ra chuỗi "Hello World".

void myMessage()

{

std::cout << "Hello World!" ;

}

myMessage();


Trong ví dụ trên, chúng ta định nghĩa một hàm có tên là myMessage().

Dấu ngoặc nhọn mở { cho biết phần đầu của hàm và dấu ngoặc nhọn đóng } cho biết phần cuối của hàm.

Hàm này in ra chuỗi "Hello World!".

Ví dụ mới

Hàm C++ sum() tính tổng của hai số nguyên.

int sum ( int a , int b )

{

return a + b ;

}

int result = sum(3, 4);

std::cout << "Tổng của 3 và 4 là " << result << std::endl;


Trong ví dụ này, chúng ta có một hàm có tên là sum(), nhận hai tham số nguyên và trả về tổng của chúng.

Sau đó, chúng ta gọi hàm sum() với các tham số là 3 và 4, và in ra kết quả bằng cách sử dụng hàm std::cout.

Câu lệnh if...else của C++

Câu lệnh if...else thực thi một số mã nếu điều kiện đúng và mã khác nếu điều kiện đó sai.

Công thức

if (condition) {
// code to be executed if condition is true;
}
else {
// code to be executed if condition is false;
}

Ví dụ

Câu lệnh if...else thực thi một số mã nếu điều kiện đúng và mã khác nếu điều kiện đó sai.

int t;

std::cin >> t;

if (t < 20) {

std::cout << "Have a good day!";

}

else {

std::cout << "Have a good night!";

}

Câu lệnh if...else if...else của C++

Câu lệnh if...else if...else thực thi các mã khác nhau cho nhiều hơn hai điều kiện.

Công thức

if (condition) {
// code to be executed if this condition is true;
}
else if (condition) {
// code to be executed if first condition is false and this condition is true;
}
else {
// code to be executed if all conditions are false;
}

Ví dụ

Câu lệnh Đầu ra "Chúc buổi sáng tốt lành!" nếu thời gian hiện tại nhỏ hơn 10 và "Chúc một ngày tốt lành!" nếu thời gian hiện tại nhỏ hơn 20. Nếu không, nó sẽ xuất ra "Chúc bạn ngủ ngon!":

int t;

std::cin >> t;

if (t < 10) {

std::cout << "Have a good morning!";

}

else if (t < 20) {

std::cout << "Have a good day!";

}

else {

std::cout << "Have a good night!";

}
