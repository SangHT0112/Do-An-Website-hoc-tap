Giới thiệu PHP

PHP là một ngôn ngữ lập trình kịch bản phía máy chủ mạnh mẽ và phổ biến. Nó đặc biệt thích hợp cho phát triển web và có thể được nhúng trực tiếp vào HTML.

PHP là viết tắt của "PHP: Hypertext Preprocessor", ban đầu được gọi là "Personal Home Page".

Tại sao nên học PHP?

  • PHP là ngôn ngữ lập trình phổ biến nhất cho phát triển web.
  • Dễ học và có cú pháp đơn giản.
  • Chạy trên nhiều nền tảng khác nhau (Windows, Linux, Unix, Mac OS X, v.v.).
  • Tương thích với hầu hết các máy chủ hiện nay (Apache, IIS, v.v.).
  • Hỗ trợ rộng rãi từ cộng đồng và nhiều tài liệu học tập.
  • Hiệu suất cao và khả năng mở rộng tốt.

Bạn có thể bắt đầu với PHP chỉ bằng cách cài đặt một máy chủ web hỗ trợ PHP như XAMPP hoặc WAMP và viết các tập tin .php đầu tiên của bạn.

Biến trong PHP


Các biến là "thùng chứa" để lưu trữ thông tin.

Tạo (Khai báo) Biến PHP

In PHP, a variable starts with the $ sign, followed by the name of the variable:


Ví dụ

$x = 5

$y = "John"


In the example above, the variable $x will hold the value 5, and the variable $y will hold the value"John".

Lưu ý: Khi bạn gán giá trị văn bản cho một biến, hãy đặt dấu ngoặc kép quanh giá trị đó.

Lưu ý: Không giống như các ngôn ngữ lập trình khác, PHP không có lệnh khai báo biến. Nó được tạo vào thời điểm bạn gán giá trị cho nó lần đầu tiên.

Hãy coi các biến như các thùng chứa để lưu trữ dữ liệu.



Tìm hiểu chuyên sâu về biến

Một biến có thể có tên ngắn (như $x và $y) hoặc tên mang tính mô tả hơn ($age , $carname , $total_volume ).

Quy Tắc Biến Trong PHP

  • Biến bắt đầu bằng ký tự $ theo sau là tên biến.
  • Tên biến phải bắt đầu bằng một chữ cái hoặc ký tự gạch dưới (_).
  • Tên biến không được bắt đầu bằng một số.
  • Tên biến chỉ có thể chứa các ký tự chữ và số và gạch dưới (A-z, 0-9, và _).
  • Tên biến phân biệt chữ hoa và chữ thường ($age và $AGE là hai biến khác nhau).

Hãy nhớ rằng tên biến PHP phân biệt chữ hoa chữ thường

Biến đầu ra

Câu lệnh echo trong PHP thường được sử dụng để xuất dữ liệu ra màn hình.

Ví dụ sau đây sẽ chỉ ra cách xuất văn bản và một biến:

Ví dụ

$txt = "W3Schools.com";

Chuỗi trong PHP


Chuỗi là một chuỗi các ký tự, chẳng hạn như "Xin chào thế giới!"

Chuỗi

Các chuỗi trong PHP được bao quanh bởi dấu ngoặc kép hoặc dấu ngoặc đơn.

Ví dụ

echo = "Hello";

echo = 'Hello';

Lưu ý Có sự khác biệt lớn giữa dấu ngoặc kép và dấu ngoặc đơn trong PHP.

Dấu ngoặc kép xử lý các ký tự đặc biệt, dấu ngoặc đơn thì không.

Chiều dài Chuỗi

The PHP strlen() function returns the length of a string.

Ví dụ

Trả về độ dài của chuỗi "Xin chào thế giới!":

echo strlen ("Hello world!");

Upper Case

Ví dụ

Hàm strtoupper() trả về chuỗi ở dạng chữ hoa:

$x = "Hello World!";

echo strtoupper($x);

Lower Case

Ví dụ

Hàm strtower() trả về chuỗi ở dạng chữ thường:

$x = "Hello World!";

echo strtower($x);

Replace String

Ví dụ

Hàm PHP str_replace() thay thế một số ký tự bằng một số ký tự khác trong chuỗi.

$x = "Hello World!";

echo str_replace($x);

Reverse a String

Ví dụ

Hàm PHP strrev() đảo ngược một chuỗi.

$x = "Hello World!";

echo strrev($x)

Toán tử trong PHP

Toán tử được sử dụng để thực hiện các thao tác trên các biến và giá trị.


PHP chia các toán tử thành các nhóm sau:

  • Toán tử số học
  • Toán tử gán
  • Toán tử so sánh

Toán tử số học

Các toán tử số học PHP được sử dụng với các giá trị số để thực hiện các phép toán số học phổ biến, chẳng hạn như cộng, trừ, nhân, v.v.

Toán Tử Tên Mô Tả Ví Dụ
+ Cộng Tổng của $x và $y $x + $y
- Trừ Sự khác biệt của $x và $y $x - $y
* Nhân Tích của $x và $y $x * $y
/ Chia Chia một giá trị cho một giá trị khác $x / $y
% Chia Lấy Dư Trả về phần dư của phép chia $x % $y
** lũy thừa Kết quả của việc nâng $x lên lũy thừa $y' $x ** $y

Toán tử gán

Toán tử gán PHP được sử dụng với các giá trị số để ghi giá trị vào một biến.

Toán tử gán cơ bản trong PHP là "=". Điều đó có nghĩa là toán hạng bên trái được đặt thành giá trị của biểu thức gán ở bên phải.

Assignment Same as... Description
x = y x = y Phép gán: toán hạng bên trái được gán giá trị của biểu thức ở bên phải
x += y x = x + y Cộng
x -= y x = x - y Trừ
x *= y x = x * y Nhân
x /= y x = x / y Chia
x %= y x = x % y Chia lấy dư

Toán tử so sánh

Toán tử so sánh PHP được sử dụng để so sánh hai giá trị (số hoặc chuỗi):

Toán tử Tên Ví dụ Kết quả
== Bằng $x == $y Trả về true nếu $x bằng $y
=== Đồng nhất $x === $y Trả về true nếu $x bằng $y, và chúng cùng kiểu dữ liệu
!= Không bằng $x != $y Trả về true nếu $x không bằng $y
<> Không bằng $x <> $y Trả về true nếu $x không bằng $y
!== Không đồng nhất $x !== $y Trả về true nếu $x không bằng $y, hoặc chúng không cùng kiểu dữ liệu
> Lớn hơn $x > $y Trả về true nếu $x lớn hơn $y
< Nhỏ hơn $x < $y Trả về true nếu $x nhỏ hơn $y
>= Lớn hơn hoặc bằng $x >= $y Trả về true nếu $x lớn hơn hoặc bằng $y
<= Nhỏ hơn hoặc bằng $x <= $y Trả về true nếu $x nhỏ hơn hoặc bằng $y
<=> Phép chấm $x <=> $y Trả về một số nguyên nhỏ hơn, bằng hoặc lớn hơn không, tùy thuộc vào $x có nhỏ hơn, bằng hoặc lớn hơn $y. Giới thiệu trong PHP 7.

Hàm trong PHP

Sức mạnh thực sự của PHP đến từ các chức năng của nó.

PHP có hơn 1000 hàm dựng sẵn và ngoài ra bạn có thể tạo các hàm tùy chỉnh của riêng mình.

Các hàm dựng sẵn trong PHP

PHP có hơn 1000 hàm dựng sẵn có thể được gọi trực tiếp, từ bên trong tập lệnh, để thực hiện một tác vụ cụ thể.

Vui lòng xem tài liệu tham khảo PHP của chúng tôi để có cái nhìn tổng quan đầy đủ về các hàm dựng sẵn PHP.

Các hàm do người dùng xác định PHP

Bên cạnh các hàm PHP tích hợp, bạn có thể tạo các hàm của riêng mình.

  • Hàm là một khối câu lệnh có thể được sử dụng nhiều lần trong chương trình.
  • Một chức năng sẽ không tự động thực thi khi tải trang.
  • Một hàm sẽ được thực thi bằng một lệnh gọi hàm.

Tạo một hàm

A user-defined function declaration starts with the keyword function, followed by the name of the function:

Ví dụ

Hàm PHP strrev() đảo ngược một chuỗi.

function myMessage() {

echo "Hello World";

}

myMessage()


Trong ví dụ của chúng tôi, chúng tôi tạo một hàm có tên myMessage().

Dấu ngoặc nhọn mở { cho biết phần đầu của mã hàm và dấu ngoặc nhọn đóng } cho biết phần cuối của hàm.

Hàm xuất ra "Hello World!".

Câu lệnh if...else của PHP

Câu lệnh if...else thực thi một số mã nếu điều kiện đúng và mã khác nếu điều kiện đó sai.

Công thức

if (condition) {
// code to be executed if condition is true;
}
else {
// code to be executed if condition is false;
}

Ví dụ

Câu lệnh if...else thực thi một số mã nếu điều kiện đúng và mã khác nếu điều kiện đó sai.

$t = date ("H");

if ($t < "20") {

echo "Have a good day!";

}

else {

echo "Have a good night!";

}

>

PHP - Câu lệnh if...elseif...else

Câu lệnh if...elseif...else thực thi các mã khác nhau cho nhiều hơn hai điều kiện.

Công thức

if (condition) {
// code to be executed if this condition is true;
}
elseif (condition) {
// code to be executed if first condition is false and this condition is true;
}
else {
// code to be executed if all conditions are false;
}

Ví dụ

Câu lệnh Đầu ra "Chúc buổi sáng tốt lành!" nếu thời gian hiện tại nhỏ hơn 10 và "Chúc một ngày tốt lành!" nếu thời gian hiện tại nhỏ hơn 20. Nếu không, nó sẽ xuất ra "Chúc bạn ngủ ngon!":

$t = date ("H");

if ($t < "10") {

echo "Have a good morning!";

}

elseif ($t < "20") {

echo "Have a good day!";

}

else {

echo "Have a good night!";

}
