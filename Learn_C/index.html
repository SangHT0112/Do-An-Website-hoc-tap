Giới thiệu ngôn ngữ C

C là một ngôn ngữ lập trình mạnh mẽ và phổ biến. Nó đặc biệt thích hợp cho việc phát triển hệ điều hành, trình biên dịch, và các hệ thống nhúng.

C được phát triển vào đầu những năm 1970 bởi Dennis Ritchie tại Bell Labs.

Tại sao nên học C?

  • C là nền tảng của nhiều ngôn ngữ lập trình hiện đại như C++, Java, và C#.
  • Cung cấp khả năng kiểm soát cao cấp đối với phần cứng và bộ nhớ.
  • Chạy trên nhiều nền tảng khác nhau (Windows, Linux, Unix, Mac OS X, v.v.).
  • Hiệu suất cao và thường được sử dụng cho các ứng dụng yêu cầu tài nguyên hạn chế.
  • Được hỗ trợ rộng rãi và có nhiều tài liệu học tập từ cộng đồng.
  • Rất phù hợp để học các khái niệm lập trình cơ bản và nâng cao.

Bạn có thể bắt đầu với ngôn ngữ lập trình C bằng cách cài đặt một trình biên dịch như GCC hoặc CLANG và viết các tập tin mã nguồn C đầu tiên của bạn.

Biến trong ngôn ngữ lập trình C

Các biến trong ngôn ngữ lập trình C là "thùng chứa" để lưu trữ thông tin.

Tạo (Khai báo) Biến trong C

Trong ngôn ngữ lập trình C, bạn cần khai báo biến trước khi sử dụng:

Ví dụ

int x = 5;

char y = 'A';


Trong ví dụ trên, biến x sẽ lưu trữ giá trị 5, và biến y sẽ lưu trữ giá trị 'A'.

Lưu ý: Khi bạn gán giá trị văn bản cho một biến trong ngôn ngữ lập trình C, bạn cần sử dụng dấu nháy đơn ('') thay vì dấu nháy kép ("").

Lưu ý: Bạn cần khai báo biến trước khi sử dụng trong ngôn ngữ lập trình C.

Hãy coi các biến trong ngôn ngữ lập trình C như các thùng chứa để lưu trữ dữ liệu.


Tìm hiểu chuyên sâu về biến trong C

Một biến trong ngôn ngữ lập trình C có thể có tên ngắn hoặc tên mang tính mô tả hơn.

Quy Tắc Biến Trong ngôn ngữ lập trình C

  • Biến trong C bắt đầu bằng ký tự chữ cái hoặc dấu gạch dưới.
  • Tên biến trong C không được bắt đầu bằng một số.
  • Tên biến trong C chỉ chứa các ký tự chữ cái, số và dấu gạch dưới (A-z, 0-9, và _).
  • Tên biến trong C không phân biệt chữ hoa và chữ thường.

Hãy nhớ rằng tên biến trong ngôn ngữ lập trình C không phân biệt chữ hoa chữ thường.

Biến đầu ra trong C

Câu lệnh printf trong ngôn ngữ lập trình C thường được sử dụng để xuất dữ liệu ra màn hình.

Ví dụ sau đây sẽ chỉ ra cách xuất văn bản và giá trị của một biến trong ngôn ngữ lập trình C:

Ví dụ

char txt = "W3Schools.com";

printf ("I love %s!", txt);

Chuỗi trong ngôn ngữ lập trình C


Chuỗi trong ngôn ngữ lập trình C là một chuỗi các ký tự, chẳng hạn như "Xin chào thế giới!"

Chuỗi

Các chuỗi trong ngôn ngữ lập trình C được bao quanh bởi dấu nháy kép.

Ví dụ

printf ("Hello");

printf ("Hello");

Lưu ý: Trong ngôn ngữ lập trình C, không có sự khác biệt giữa dấu nháy kép và dấu nháy đơn.

Chiều dài Chuỗi

Trong ngôn ngữ lập trình C, bạn có thể sử dụng hàm strlen() để trả về độ dài của một chuỗi.

Ví dụ

Trả về độ dài của chuỗi "Xin chào thế giới!":

printf ("%d", strlen("Hello world!"));

Upper Case

Ví dụ

Hàm toupper() trả về chuỗi ở dạng chữ hoa:

char x[] = "Hello World!";

printf ("%s", strupr(x));

Lower Case

Ví dụ

Hàm tolower() trả về chuỗi ở dạng chữ thường:

char x[] = "Hello World!";

printf ("%s", strlwr(x));

Toán tử trong C

Toán tử được sử dụng để thực hiện các phép toán trên biến và giá trị.


C in phân loại các toán tử thành các nhóm sau:

  • Toán tử số học
  • Toán tử gán
  • Toán tử so sánh

Toán tử số học

Các toán tử số học trong C được sử dụng với các giá trị số để thực hiện các phép toán số học phổ biến như cộng, trừ, nhân, v.v.

Toán Tử Tên Mô Tả Ví Dụ
+ Cộng Tổng của x và y x + y
- Trừ Hiệu của x và y x - y
* Nhân Tích của x và y x * y
/ Chia Chia x cho y x / y
% Chia Lấy Dư Trả về phần dư của phép chia x % y

Toán tử gán

Toán tử gán trong C được sử dụng với các giá trị số để ghi giá trị vào một biến.

Toán tử gán cơ bản trong C là "=". Điều đó có nghĩa là toán hạng bên trái được gán giá trị của biểu thức ở bên phải.

Toán tử gán Tương đương... Mô tả
x = y x = y Toán tử gán: toán hạng bên trái được gán giá trị của biểu thức ở bên phải
x += y x = x + y Cộng
x -= y x = x - y Trừ
x *= y x = x * y Nhân
x /= y x = x / y Chia
x %= y x = x % y Modulus

Toán tử so sánh

Toán tử so sánh trong C được sử dụng để so sánh hai giá trị (số hoặc chuỗi):

Toán tử Tên Ví dụ Kết quả
== Bằng $x == $y Trả về true nếu $x bằng $y
!= Không bằng $x != $y Trả về true nếu $x không bằng $y
< Nhỏ hơn $x < $y Trả về true nếu $x nhỏ hơn $y
> Lớn hơn $x > $y Trả về true nếu $x lớn hơn $y
<= Nhỏ hơn hoặc bằng $x <= $y Trả về true nếu $x nhỏ hơn hoặc bằng $y
>= Lớn hơn hoặc bằng $x >= $y Trả về true nếu $x lớn hơn hoặc bằng $y

Hàm trong C

Điểm mạnh thực sự của C đến từ khả năng sử dụng hàm.

C có một số hàm tiêu chuẩn được tích hợp sẵn và bạn cũng có thể tự định nghĩa các hàm tùy chỉnh của riêng mình.

Các hàm tiêu chuẩn trong C

Cung cấp hơn 1000 hàm tiêu chuẩn có thể gọi trực tiếp từ bên trong chương trình để thực hiện các tác vụ cụ thể.

Vui lòng tham khảo tài liệu C của chúng tôi để có cái nhìn tổng quan đầy đủ về các hàm tiêu chuẩn của C.

Các hàm do người dùng xác định trong C

Ngoài các hàm tiêu chuẩn, bạn cũng có thể tự định nghĩa các hàm của riêng mình trong C.

  • Hàm là một khối lệnh có thể tái sử dụng nhiều lần trong chương trình.
  • Một hàm sẽ không tự động thực thi khi chương trình được tải.
  • Một hàm sẽ được thực thi thông qua lệnh gọi hàm.

Tạo một hàm

Định nghĩa hàm do người dùng bắt đầu bằng từ khóa void hoặc kiểu dữ liệu của giá trị trả về (nếu có), tiếp theo là tên của hàm:

Ví dụ

Hàm C myMessage() in ra chuỗi "Hello World".

void myMessage()

{

printf ( "Hello World!\n" );

}

myMessage();


Trong ví dụ trên, chúng ta định nghĩa một hàm có tên là myMessage().

Dấu ngoặc nhọn mở { cho biết phần đầu của hàm và dấu ngoặc nhọn đóng } cho biết phần cuối của hàm.

Hàm này in ra chuỗi "Hello World!".

Ví dụ mới

Hàm C sum() tính tổng của hai số nguyên.

int sum ( int a , int b )

{

return a + b ;

}

int result = sum(3, 4);

printf ( "Tổng của 3 và 4 là %d\n", result );


Trong ví dụ này, chúng ta có một hàm có tên là sum(), nhận hai tham số nguyên và trả về tổng của chúng.

Sau đó, chúng ta gọi hàm sum() với các tham số là 3 và 4, và in ra kết quả bằng cách sử dụng hàm printf().

Câu lệnh if...else của C

Câu lệnh if...else thực thi một số mã nếu điều kiện đúng và mã khác nếu điều kiện đó sai.

Công thức

if (condition) {
// code to be executed if condition is true;
}
else {
// code to be executed if condition is false;
}

Ví dụ

Câu lệnh if...else thực thi một số mã nếu điều kiện đúng và mã khác nếu điều kiện đó sai.

int t;

scanf ("%d", &t);

if (t < 20) {

printf ("Have a good day!");

}

else {

printf ("Have a good night!");

}

Câu lệnh if...else if...else của C

Câu lệnh if...else if...else thực thi các mã khác nhau cho nhiều hơn hai điều kiện.

Công thức

if (condition) {
// code to be executed if this condition is true;
}
else if (condition) {
// code to be executed if first condition is false and this condition is true;
}
else {
// code to be executed if all conditions are false;
}

Ví dụ

Câu lệnh Đầu ra "Chúc buổi sáng tốt lành!" nếu thời gian hiện tại nhỏ hơn 10 và "Chúc một ngày tốt lành!" nếu thời gian hiện tại nhỏ hơn 20. Nếu không, nó sẽ xuất ra "Chúc bạn ngủ ngon!":

int t;

scanf ("%d", &t);

if (t < 10) {

printf ("Have a good morning!");

}

else if (t < 20) {

printf ("Have a good day!");

}

else {

printf ("Have a good night!");

}
