Hướng dẫn CSS

CSS là ngôn ngữ chúng ta sử dụng để tạo kiểu cho một tài liệu HTML.


CSS mô tả cách các phần tử HTML sẽ được hiển thị.


Hướng dẫn này sẽ dạy cho bạn CSS từ cơ bản đến nâng cao.


Tham khảo CSS

Tại đây, bạn sẽ tìm thấy các tham chiếu CSS hoàn chỉnh của tất cả các thuộc tính và bộ chọn với cú pháp, ví dụ, hỗ trợ trình duyệt và hơn thế nữa.

CSS Introduce

CSS là ngôn ngữ chúng ta sử dụng để tạo kiểu cho một trang Web.

CSS là gì?

  • CSS là viết tắt của Cascading Style Sheets
  • CSS mô tả cách các phần tử HTML được hiển thị trên màn hình, giấy, hoặc trong các phương tiện truyền thông khác
  • CSS tiết kiệm rất nhiều công việc. Nó có thể kiểm soát bố cục của Nhiều trang web cùng một lúc
  • Biểu định kiểu bên ngoài được lưu trữ trong tệp CSS

CSS Colors

CSS hỗ trợ 140+ tên màu, giá trị HEX, giá trị RGB, RGBA giá trị, giá trị HSL, giá trị HSLA và độ mờ.


Ví dụ sau đây xác định các màu RGBA khác nhau:


Red {background-color:rgba(255,0,0,0.3);}

Green {background-color:rgba(0,255,0,0.3);}

Blue {background-color:rgba(0,0,255,0.3);}

Grey {background-color:rgba(192,192,192,0.3);}

Yellow {background-color:rgba(255,255,0,0.3);}

Cerise {background-color:rgba(255,0,255,0.3);}
